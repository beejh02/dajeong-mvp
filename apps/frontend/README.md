# Dajung Frontend

Next.js App Router 기반의 Dajung 키오스크/관리자 프론트엔드입니다.

## Scripts

- `pnpm dev`: Next.js 개발 서버 실행
- `pnpm build`: 프로덕션 빌드 검증
- `pnpm start`: 빌드된 앱 실행
- `pnpm lint`: ESLint 실행
- `pnpm verify:next`: Next.js 구조 검증

## Routes

- `/`: 키오스크/관리자 선택 홈
- `/admin`: 관리자 페이지
- `/kiosk-a`: A기업 키오스크
- `/kiosk-b`: B기업 키오스크
