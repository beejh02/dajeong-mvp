# 프로젝트 의도

## 왜 다정을 만드는가

기업마다 다른 디지털 서비스 UI는 사용자에게 학습 부담을 준다.

특히 키오스크처럼 현장에서 빠르게 조작해야 하는 환경에서는 UI 차이가 곧 접근성 문제로 이어질 수 있다.

다정은 사용자가 복잡한 UI를 직접 익히지 않아도, AI에게 자연어로 요청하면 필요한 작업을 대신 진행할 수 있는 가능성을 보여주기 위한 MVP이다.

## 다정은 무엇인가

다정은 키오스크 앱이 아니다.

다정은 여러 기업 서비스와 연결될 수 있는 AI 기반 중간 플랫폼이다.

이번 MVP에서 키오스크는 다정의 가능성을 보여주기 위한 대표 데모일 뿐이다.
