# 키오스크 데모

## 목적

키오스크 데모는 기업별 UI/UX 차이가 사용자 경험에 어떤 영향을 주는지 보여주기 위한 기능이다.

## A기업 키오스크

- Vertical UI
- 세로 방향 메뉴 탐색
- 상단 또는 좌측 카테고리 중심 구조
- 사용자가 위에서 아래로 흐름을 따라가도록 설계

## B기업 키오스크

- Horizontal UI
- 가로 방향 메뉴 탐색
- 넓은 화면 기반의 메뉴 배치
- A기업과 다른 조작 경험을 제공

## 핵심 비교 포인트

- 메뉴 탐색 방식 차이
- 장바구니 위치 차이
- 옵션 선택 방식 차이
- 주문 완료까지의 흐름 차이
- 사용자 혼란 가능성
