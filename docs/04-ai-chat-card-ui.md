# 다정 AI 채팅 및 카드 UI

## 목적

다정 AI는 사용자의 자연어 요청을 이해하고, 주문이나 조회 같은 작업을 도와준다.

## 채팅 UI

사용자는 텍스트 또는 음성 기반으로 요청할 수 있다.

예시:

- 불고기버거 세트 하나 주문해줘
- 포인트 얼마나 있어?
- 제일 인기 있는 메뉴 추천해줘
- 주문 내역 보여줘

## 카드 UI

AI 응답은 단순 텍스트만 사용하지 않고 카드 형태의 구조화된 응답을 포함한다.

## 카드 예시

- 메뉴 추천 카드
- 주문 요약 카드
- 결제 확인 카드
- 포인트 안내 카드
- 액션 카드
- 관리자 요약 카드

## 카드 UI를 사용하는 이유

카드는 사용자가 긴 텍스트를 읽지 않아도 핵심 정보를 빠르게 이해할 수 있게 한다.

또한 주문, 결제, 포인트처럼 구조화된 정보를 표현하기 좋다.
