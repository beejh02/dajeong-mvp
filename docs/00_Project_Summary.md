# Dajeong MVP 프로젝트 요약

## 한 줄 정의

다정(Dajeong)은 기업마다 다른 키오스크와 예약/결제 화면을 하나의 자연어 대화 흐름으로 감싸는 접근성 중심 AI 플랫폼 MVP입니다.

## 문제

사용자는 매장, 브랜드, 서비스마다 다른 화면 구조를 매번 학습해야 합니다. 같은 햄버거 주문이라도 메뉴 위치, 옵션 선택, 결제 버튼, 주문 확인 방식이 달라지고, 교통/예약 서비스까지 확장되면 부담이 더 커집니다.

## 해결 방식

다정은 사용자의 자연어 요청을 받아 의도와 필요한 정보를 정리하고, 기업별 API 또는 MCP Tool을 호출해 실제 작업을 대신 수행합니다. 사용자는 다정의 대화형 UI만 익히면 되고, 기업별 UI 차이는 시스템 내부 adapter와 MCP 경계가 흡수합니다.

## MVP 발표 메시지

MVP는 "대화형 UI + MCP 호출 경계 + 관리자 검증 화면"을 보여줍니다. 단순 챗봇이 아니라 외부 기업 기능을 도구 호출로 연결하는 플랫폼 구조임을 관리자 페이지의 주문 데이터와 MCP 로그로 증명합니다.

## 대표 시연 문장

```text
늘 먹던 햄버거 하나 주문해줘. 오이는 빼줘.
```

## 시연 계정

```text
일반 사용자: user1 / user1234
관리자: admin / dajeong
```
