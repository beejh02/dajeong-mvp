# MVP 범위

## 포함하는 기능

- A기업 키오스크 화면
- B기업 키오스크 화면
- 다정 AI 채팅 화면
- AI 카드형 응답
- 주문 생성 흐름
- 메뉴 조회 흐름
- 관리자 페이지
- Backend API
- MCP Server 기본 구조

## 제외하는 기능

- 실제 카드 결제
- 실제 Toss Payments 운영 결제
- 실제 Korail 예매
- 실제 기업 API 연동
- 사용자 인증 고도화
- 운영용 배포 안정화
- 복잡한 권한 관리

## 데모 기준

A기업과 B기업은 서로 다른 UI 구조를 가진다.

- A기업: Vertical UI
- B기업: Horizontal UI

두 키오스크는 같은 목적을 수행하지만, 사용자가 느끼는 UX 차이를 보여주기 위해 UI 구조를 다르게 설계한다.
